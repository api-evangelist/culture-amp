# Culture Amp GraphQL API

Culture Amp is an employee experience platform. The API covers employee surveys, engagement scores, performance reviews, goal tracking, 1-on-1 meetings, action plans, and workforce analytics for HR teams.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Culture Amp REST API
**Documentation:** https://docs.api.cultureamp.com/docs/resources-getting-started
